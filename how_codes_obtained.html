How the Codes Were Obtained

1. Based on Objectives

The codes were written and adapted to align directly with the research objectives:

  • i. Build a transfer learning model for image classification.
  • ii. Evaluate it with accuracy, precision, recall, and F1-score.
  • iii. Compare it with a conventional machine learning baseline (SVM with HOG).

Therefore, each section of code was chosen to implement exactly these objectives.

2. Using Standard Frameworks

The snippets come from widely used, open-source libraries to ensure reproducibility and accessibility:

  • TensorFlow/Keras → for MobileNetV2 feature extraction & training.
  • PyTorch → for fine-tuning MobileNetV2 with flexibility.
  • Scikit-learn → for building the SVM baseline and evaluation metrics.
  • Skimage (hog) → for extracting handcrafted HOG features.

This structured selection of frameworks ensures that the code is grounded in standard, peer-reviewed practices in machine learning and computer vision.
